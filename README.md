> Originally adapted into Gambella Policy Lab, a civic platform focused on governance, refugee dynamics, public policy, and regional development analysis in Gambella, Ethiopia.

## Features

- Governance and institutional profiles
- Refugee and demographic trend analysis
- Historical timeline of major Gambella events
- Head-to-head comparison tool
- Methodology page explaining the scoring system
- Policy blog and analysis articles
- Contact and civic engagement tools

## Project Structure

```txt
app/
├── candidats/[slug]/  # Governance profile pages
├── comparateur/       # Head-to-head comparison

├── timeline/          # Historical timeline of Gambella events
